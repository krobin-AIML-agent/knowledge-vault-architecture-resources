# Free & Open-Source SDKs and ADKs (AI Agents)
A curated list of 8 agent frameworks with direct links and one-line summaries.

| # | Name | Link | Summary |
|---|------|------|---------|
| 1 | **Agent Development Kit (ADK)** (Google) | https://google.github.io/adk-docs/ | Code-first framework for building and orchestrating multi-agent systems (Python/Go/Java), deployment-agnostic. |
| 2 | **OpenAI Agents SDK** | https://openai.github.io/openai-agents-python/ | Lightweight primitives (Agents, Handoffs, Guardrails) for creating agent workflows with tools and LLMs. |
| 3 | **LangChain** | https://github.com/langchain-ai/langchain | Framework for connecting LLMs, tools, memory, retrieval; strong agent and workflow support. |
| 4 | **Microsoft Agent Framework** | https://learn.microsoft.com/en-us/agent-framework/overview/agent-framework-overview | Open-source agent framework merging Semantic Kernel + AutoGen for multi-agent orchestration (Python/.NET). |
| 5 | **Strands Agents** (AWS) | https://aws.amazon.com/blogs/opensource/introducing-strands-agents-an-open-source-ai-agents-sdk/ | Open-source SDK from AWS; model-driven agent creation for simple-to-complex workloads. |
| 6 | **AutoGen** (Microsoft) | https://github.com/microsoft/autogen | Multi-agent framework where agents collaborate via conversations; high-level abstractions. |
| 7 | **Open Agent Platform** (LangChain team) | https://github.com/langchain-ai/open-agent-platform | No-code + code platform for building, managing, and deploying AI agents. |
| 8 | **AgentOps** | https://github.com/AgentOps-AI/agentops | Open-source SDK for observability: monitoring, debugging, and analyzing agent performance. |
