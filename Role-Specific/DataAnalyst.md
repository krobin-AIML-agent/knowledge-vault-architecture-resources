# ROLE 4: DATA ANALYST  
**Role Context:** Data Analyst  
**Primary Focus:** Reporting, dashboards, business insights, ad-hoc analysis  
**Daily Tasks:** SQL analysis, cleaning datasets, building dashboards, generating insights  
**Output:** Reports, KPIs, dashboards, business recommendations

---

# MUST-HAVE (Primary): 80/20 Core

## 1. SQL for Analytics  
**System Alignment:** Analytics Layer → Query & Aggregation  
**Task Frequency:** Daily  

### Your 20%  
- **SELECT / WHERE / GROUP BY / HAVING**  
- **JOINs** (inner, left, full)  
- **Window functions**:  
  https://www.postgresql.org/docs/current/tutorial-window.html  
- **CTEs**  
- **Aggregation patterns (SUM, AVG, COUNT)**  

### Tools  
- **PostgreSQL**: https://www.postgresql.org/docs  
- **BigQuery SQL**: https://cloud.google.com/bigquery/docs/reference/standard-sql  
- **Snowflake SQL**: https://docs.snowflake.com/en/sql-reference  

### What You *Don't* Need  
- Deep DB administration  
- Query planner internals  

### Pattern Alignment  
- Filter → Aggregate → Visualize  
- Rolling windows  
- Ranking/filter patterns  

---

## 2. Spreadsheet Proficiency  
**System Alignment:** BI Layer → Ad-hoc Analysis  
**Task Frequency:** Daily  

### Your 20%  
- **Excel**: https://support.microsoft.com/excel  
- **Google Sheets**: https://support.google.com/docs  
- Formulas (VLOOKUP, INDEX/MATCH, SUMIF, COUNTIF)  
- Pivot Tables  
- Conditional formatting  

### What You *Don't* Need
