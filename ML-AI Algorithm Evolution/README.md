# AI/ML Knowledge Vault

This repository brings together the core foundations of machine intelligence into one organized system.  
It combines **AI evolution**, **algorithm behavior**, **essential math**, **data science logic**, and **computer science fundamentals** all in a way that is clear, structured, and easy to study.

The goal is simple:  
**Put everything needed to understand modern AI in one place.**

---

## What’s Inside

### 1. Machine Intelligence Evolution Map  
A top-down view of how AI developed over time and why each shift happened.

### 2. Algorithm Characteristics  
Shows how different algorithms work, their strengths, limits, and when to use them.

### 3. Key Math for Data Science  
A compact reference for the math behind ML (linear algebra, probability, optimization, calculus).

### 4a. Data Science & Algorithms  
Explains workflows, decisions, and practical logic behind real ML work.

### 4b. Computer Science Foundations  
Covers computation, complexity, memory, structures, and system concepts relevant to ML.
